------------------------------------------------
A debt settling and tracking platform built to work in the Solana Web3 ecosystem.


### To Setup:
1) add firebase-service.json to config/
2) `go mod tidy`
3) `go generate ./ent`
4) `go run .`

### Dev Environment
1) change `ENV_TYPE` in the .env file to dev to enable dev environment

### Prod Environment
1) change `ENV_TYPE` in the .env file to prod to enable prod environment
